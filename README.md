# ejercicio1-dom-frontend

Objetivo:
Modificar el DOM de una pagina

Instrucciones:
1. Con base en el diseño en Figma del [Login](https://www.figma.com/design/p9wHcBw7RJDWXXIrtlv5gV/Ejercicio---Error-en-Login---JS?node-id=0-1&p=f&t=QrmIPRkNe8Cbd09v-0), actualiza la pagina agregando dinamicamente los mensaje de error en pantalla. Sigue los pasos en el código.
2. Si el usuario hace clic en "Olvide mi contrasña" debe aparecer una ventana como en la vista de Figma que superponga a todo el formulario, de forma que el usuario deba ingresar su cuenta de correo para recibir un link para restrablecer su contraseña. Puedes utilizar la propiedad de CSS z-index: 9999  para lograrlo y observa que detrás de la ventana hay un fondo gris que hace que el contenido debajo se oscuresca.
